Cursos Disponibles

Aprende a estructurar páginas web con HTML5. Este curso incluye:
  • Estructura básica de una página web
  • Etiquetas semánticas: header, nav, main, footer
  • Creación de formularios y tablas
  • Enlaces, listas, imágenes y multimedia
  • Buenas prácticas de accesibilidad

Duración: 4 semanas

Precio: $50.000

Diseña y estiliza páginas web modernas con CSS3. Contenidos:
  • Estilos básicos: colores, fuentes, bordes y fondos
  • Box model y posicionamiento
  • Flexbox y Grid Layout
  • Animaciones y transiciones
  • Responsive Design con Media Queries

Duración: 4 semanas

Precio: $50.000

Agrega interactividad a tus sitios con JavaScript. Este curso abarca:
  • Variables, tipos de datos y operadores
  • Funciones y estructuras de control (if, switch, loops)
  • Manejo del DOM (Document Object Model)
  • Eventos, validación de formularios
  • Introducción a objetos, arrays y funciones flecha
  • Uso de librerías básicas como SweetAlert y Bootstrap JS

Duración: 5 semanas

Precio: $60.000

Aprende programación desde cero con Python. El curso incluye:
  • Sintaxis básica y uso de variables
  • Condicionales y bucles (if, for, while)
  • Funciones definidas por el usuario
  • Manejo de archivos y errores
  • Introducción a estructuras de datos (listas, tuplas, diccionarios)
  • Mini proyectos prácticos (calculadoras, juegos, automatización básica)

Duración: 6 semanas

Precio: $70.000
